# Controle de Efetivo PWA

Sistema de gestão de presença e ausências de funcionários.

## Tecnologias Utilizadas

Este projeto foi desenvolvido com:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- PWA (Progressive Web App)

## Como Executar Localmente

Requisitos: Node.js e npm instalados - [instalar com nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

Siga estes passos:

```sh
# Passo 1: Clone o repositório
git clone <YOUR_GIT_URL>

# Passo 2: Navegue até o diretório do projeto
cd efetivo-pwa

# Passo 3: Instale as dependências
npm install

# Passo 4: Inicie o servidor de desenvolvimento
npm run dev
```

## Scripts Disponíveis

- `npm run dev` - Inicia o servidor de desenvolvimento
- `npm run build` - Cria build de produção
- `npm run preview` - Visualiza o build de produção
- `npm run lint` - Executa o linter
- `npm run test` - Executa os testes
- `npm run test:watch` - Executa os testes em modo watch

## Deploy

Para fazer o deploy da aplicação, você pode utilizar serviços como:

- Vercel
- Netlify
- GitHub Pages
- Railway
- Render

Basta conectar seu repositório e configurar o comando de build como `npm run build`.

## PWA Features

A aplicação está configurada como PWA (Progressive Web App) e pode ser instalada em dispositivos móveis e desktops, funcionando offline após a primeira visita.
